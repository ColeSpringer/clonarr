Edition Tags & Plex
The recommended naming includes {edition-{Edition Tags}} which tells Plex to recognize movie editions (e.g. Director's Cut, Theatrical) and display them in the interface.
However, if you keep two copies of a movie with different editions in a merged library (e.g. 1080p Director's Cut + 2160p Theatrical), Plex will show them as two separate items. Enable the toggle to use {Edition Tags} instead — editions still appear in the filename but Plex treats all copies as one entry.
Movie File Format
How movie files are named. Includes quality, audio, HDR, and release group info. Plex/Emby/Jellyfin variants add media server IDs for better matching.
Standard Recommended Copy Apply
The recommended naming scheme. Includes all non-recoverable information in brackets.
Plex (IMDb) Copy Apply
For the Plex agent. IMDb IDs are very stable and rarely change.
Plex (TMDb) Copy Apply
For the Plex agent. TMDb is more aligned with Radarr but IDs can occasionally change.
Original Title Copy Apply
Uses the release title as-is. Prevents download loops from title/content mismatches, but gives less control over naming.
