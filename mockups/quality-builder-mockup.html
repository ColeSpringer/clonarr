Placement: This editor appears inline in the profile builder settings panel, below the current Quality Preset dropdown. When a TRaSH preset is selected, the editor is pre-populated with that preset's quality configuration. "Custom..." option in the preset dropdown opens this editor for manual configuration.

No drag-and-drop required: Arrow buttons handle all reordering. This keeps the implementation simple without external dependencies.

Data format: The editor produces the same items[] array used by TRaSH profiles — no backend changes needed.
