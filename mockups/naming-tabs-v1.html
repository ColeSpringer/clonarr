TRaSH recommended naming schemes. Adding non-recoverable info (quality, release group, edition) to filenames prevents duplicate downloads and re-import loops. Media servers show titles — not filenames — so longer names don't affect viewing. Both IMDb and TMDb IDs work, but TMDb is usually better as it guarantees a match — IMDb only works if the TMDb entry has the correct association.

Edition Tags
Plex naming uses {edition-{Edition Tags}} so Plex recognizes editions (Director's Cut, Theatrical, etc.) and shows them in the UI. If you keep multiple copies with different editions in a merged library (e.g. 1080p Director's Cut + 2160p Theatrical), Plex will show them as separate items.
Enable single-entry mode to replace {edition-{Edition Tags}} with {Edition Tags} — editions still appear in the filename but Plex treats all copies as one library entry.
